Wasmer Java Runtime Demo
========================

### Features

* Run wasm
* Memory
* Greet

### Settings on Mac
If you get "java.lang.UnsatisfiedLinkError: no wasmer_jni in java.library.path" error,
please copy libwasmer_jni.dylib to  ~/Library/Java/Extensions/
